    性能优化¶

    • 关闭防火墙(iptables selinux)
    • 检查开机启动项
    • 调整文件描述符限制
      1. vim /etc/security/limits.conf 写入* - nofile 65535
      2. ulimit -SHn 65535 加入/etc/rc.local
